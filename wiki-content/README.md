## Content Sync

This content is automatically synchronized between the main docs/ and the GitHub Wiki:

1. **From main docs to wiki**:
   - Documentation from `docs/` is formatted for wiki usage
   - Technical details are simplified for end users
   - Examples and guides are enhanced for community use

2. **From wiki to main docs**:
   - Community contributions are reviewed and merged
   - User feedback is incorporated into official documentation
   - Wiki navigation is maintained in this directory

The wiki differs from the main `docs/` directory in several ways:
