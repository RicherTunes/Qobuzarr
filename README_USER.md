# Qobuzarr - Lidarr Plugin for Qobuz

High-quality music indexer and download client for Qobuz streaming service.

## Features

- **High-fidelity audio support** up to 24-bit/192kHz FLAC
- **Email/password and token authentication** methods  
- **Configurable concurrent downloads** with queue management
- **Advanced search filtering** and metadata integration
- **Secure credential management** via environment variables

## Quick Install

1. Copy `Lidarr.Plugin.Qobuzarr.dll` to your Lidarr plugins directory
2. Set environment variables: `QOBUZ_APP_ID` and `QOBUZ_APP_SECRET`  
3. Restart Lidarr
4. Configure indexer and download client in Lidarr settings

## Requirements

- Lidarr v10.0+
- Active Qobuz subscription
- .NET 6.0 runtime
